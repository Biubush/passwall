## 简介

一个免费的科学上网订阅链接，提供稳定、高速、免费的科学上网

## 使用

从[网站](https://passwall.biubush.cn)复制对应软件的链接去对应软件里导入订阅就行

> 注：只能使用订阅中的[普通]级别订阅，VIP级别未解锁，使用了也不能上网

## 注意事项

订阅链接24小时更新一次，所以24小时后你上次获取的订阅就失效了，更新订阅是解决不了问题的。

我建议的方法是，把上次的订阅删除，重新导入一遍，就又能用了。
